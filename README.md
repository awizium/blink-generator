# Blink Generator

Генератор изображений и чат через API Blink.

## Что это

Навайбкоденный веб-интерфейс для генерации картинок через Gemini и общения с AI-моделями.
Работает через API ключ от blink.new.
Один HTML файл, без зависимостей, всё локально в браузере.

## Быстрый старт
```bash
    git clone https://github.com/awizium/blink-generator.git
    cd blink-generator
    start.bat
```

### Что нужно

- Python 3 (нужен только для локального сервера, обычно уже установлен)
- Любой браузер (Chrome, Firefox, Edge)
- API ключ от blink.new

## Как пользоваться

1. Запустите start.bat — откроется браузер
2. Справа в настройках нажмите + Новый рядом с API Ключ
3. Введите название и ваш ключ от Blink
4. Выберите модель (для картинок — вкладка Картинки)
5. Напишите промпт и нажмите Enter

## Где хранятся данные

Всё локально в браузере:

- Настройки и ключи — localStorage
- Картинки — IndexedDB (любой размер)

Ничего не уходит на сторонние серверы. Запросы идут напрямую к blink.new.
