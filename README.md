# Blink Generator

Веб-интерфейс для генерации картинок и общения с AI-моделями через API blink.new.
Без зависимостей, всё локально в браузере.

## Быстрый старт

    git clone https://github.com/awizium/blink-generator.git

### Что нужно

- Браузер (Chrome, Firefox, Edge)
- API ключ от blink.new
- Python
- Git (опционально, для автообновления)

## Запуск

### Windows

Запустите `start.bat`. Откроется браузер с интерфейсом.

### macOS / Linux

    chmod +x start.sh
    ./start.sh

### Вручную

    python -m http.server 8080

Затем откройте `http://localhost:8080/index.html`

## Возможности

- 12 встроенных пресетов стилей (аниме, фотореализм, акварель и др.)
- Галерея всех сгенерированных изображений с поиском по промптам
- Экспорт/импорт всех данных или отдельного чата
- Горячие клавиши (Ctrl+Enter, Ctrl+G, Ctrl+Shift+N и др.)
- 15 цветовых тем
- Swipe-навигация между вариантами генерации
- Индикатор размера хранилища
- PWA — можно установить как приложение

## Горячие клавиши

| Комбинация | Действие |
|---|---|
| Enter | Отправить сообщение |
| Shift+Enter | Новая строка |
| Ctrl+Enter | Отправить из любого места |
| Ctrl+Shift+N | Новый чат |
| Ctrl+G | Открыть/закрыть галерею |
| Ctrl+/ | Фокус на поле ввода |
| Ctrl+Z | Отмена действия |
| Esc | Закрыть окна |

## Где хранятся данные

- Настройки и ключи — localStorage
- Картинки — IndexedDB

Ничего не уходит на сторонние серверы. Запросы идут напрямую к API blink.new.
